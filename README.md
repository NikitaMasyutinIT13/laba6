Требования:

Цель: МЕТОД

Доступна во время исполнения

Не имеет свойств

Алгоритм решения:

Создать аннотацию @Invoke с целевым типом METHOD и retention RUNTIME

Разработать класс InvokeDemo с методами, помеченными аннотацией

Реализовать обработчик InvokeHandler, который через Reflection:

Получает все методы класса

Проверяет наличие аннотации @Invoke
